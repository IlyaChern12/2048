# 2048 Game

Интерактивная игра 2048, реализованная с использованием JavaScript, HTML и CSS, выполненная в рамках Лабораторной работы №3 по Web-программированию.

Все элементы игры, включая игровое поле, плитки и кнопки управления, создаются и управляются через DOM API. В приложении реализованы движения плиток во всех четырёх направлениях, объединение одинаковых значений, отображение текущего счёта, возможность отмены хода, начало новой игры и сохранение текущей игровой сессии в localStorage, что позволяет продолжить игру после перезагрузки страницы. Пользователь может сохранять свои результаты в таблицу лидеров и просматривать топ-10 рекордов. Адаптивный дизайн обеспечивает удобное отображение на разных устройствах, включая десктопы и мобильные устройства.
